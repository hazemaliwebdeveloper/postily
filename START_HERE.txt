================================================================================
                           FASTEST WAY TO START
================================================================================

1. Make sure Docker Desktop is installed and running
   - Windows Start Menu > Search "Docker Desktop" > Click to launch

2. Double-click this file in the project root:
   setup-local.bat

3. Wait for all services to start (10-15 seconds)

4. Open in browser:
   Frontend:  http://localhost:4200
   Backend:   http://localhost:3000

================================================================================
                         MANUAL STARTUP (CLI)
================================================================================

1. Open Terminal/PowerShell in project root

2. Run these commands in order:
   docker compose -f docker-compose.dev.yaml up -d
   pnpm install (first time only)
   pnpm run dev

3. Application will be ready at:
   http://localhost:4200
